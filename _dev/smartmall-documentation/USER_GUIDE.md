## Getting Started

### Accessing Smart Mall

**Web:** Visit the Smart Mall website in any modern browser  
**Mobile:** Download the Smart Mall APK and install on Android devices

### First Time Setup

1. Open Smart Mall
2. Click **Register** or **Sign Up with Google**
3. Complete your profile
4. Start shopping!

## Account Management

### Registration

**Option 1: Email Registration**
1. Click **Register**
2. Enter: Full Name, Email, Password
3. Click **Create Account**
4. Verify your email (check inbox/spam)

**Option 2: Google Sign-In**
1. Click **Sign in with Google**
2. Select your Google account
3. Grant permissions
4. Account created automatically

### Login

1. Click **Login**
2. Enter email and password OR use Google Sign-In
3. Click **Sign In**

### Password Recovery

1. Click **Forgot Password?**
2. Enter your email
3. Check email for reset link
4. Click link and enter new password
5. Login with new password

### Profile Management

1. Click profile icon (top right)
2. View/edit: Name, Email, Phone, Address
3. Click **Save Changes**

## Shopping

### Browse Products

**By Category:**
- Click category cards: Fashion, Electronics, Home, Beauty
- View all products in that category

**Search:**
- Type product name in search bar
- Results appear instantly
- Click product to view details

**Filter & Sort:**
- Filter by price range
- Sort by: Newest, Price (Low/High), Name
- Apply filters to refine results

### Product Details

**View Information:**
- Product images (swipe for multiple images)
- Name, price, category
- Stock availability
- Description

**Add to Cart:**
1. Select quantity (use +/- buttons)
2. Click **Add to Cart**
3. Continue shopping or **View Cart**

### Shopping Cart

**Manage Cart:**
- View all items
- Change quantities (+/-)
- Remove items (trash icon)
- See subtotal

**Proceed to Checkout:**
- Review cart total
- Click **Proceed to Checkout**

## Orders & Payments

### Checkout Process

1. **Shipping Information:**
   - Enter: Name, Email, Phone, Address
   - All fields required

2. **Review Order:**
   - Verify items and quantities
   - Check shipping address
   - View total amount

3. **Payment:**
   - Select payment method (Chapa)
   - Click **Place Order**
   - Redirected to payment gateway

4. **Complete Payment:**
   - Enter payment details
   - Confirm transaction
   - Return to Smart Mall

### Order Confirmation

After successful payment:
- Order number displayed
- Email confirmation sent
- Order appears in Order History

### Track Orders

1. Click profile icon → **My Orders**
2. View all orders with:
   - Order number
   - Date
   - Items
   - Total
   - Status (Pending, Processing, Completed, Cancelled)
3. Click order for details

### Order Status

- **Pending:** Payment processing
- **Processing:** Being prepared for shipment
- **Completed:** Delivered
- **Cancelled:** Order cancelled

### Features

**All web features available:**
- Browse products
- Search & filter
- Shopping cart
- Checkout
- Order tracking
- Google Sign-In

**Mobile-Specific:**

## Troubleshooting

### Cannot Login

**Issue:** "Invalid credentials" error  
**Solution:**
- Check email/password spelling
- Use **Forgot Password** to reset
- Try Google Sign-In instead

### Cannot Add to Cart

**Issue:** Add to Cart button not working  
**Solution:**
- Check stock availability
- Login to your account
- Refresh page
- Clear browser cache

### Payment Failed

**Issue:** Payment not processing  
**Solution:**
- Check payment details
- Verify sufficient funds
- Try again (order still pending)
- Contact support if problem persists

### Order Not Showing

**Issue:** Completed order not in history  
**Solution:**
- Wait 5 minutes and refresh
- Check email for order confirmation
- Verify you're logged into correct account
- Contact admin if issue persists

### Mobile App Issues

**Issue:** App not installing  
**Solution:**
- Enable "Unknown Sources" in Settings → Security
- Download APK again (may be corrupted)
- Check Android version (5.0+ required)

**Issue:** App crashing  
**Solution:**
- Clear app data: Settings → Apps → Smart Mall → Clear Data
- Reinstall app
- Check internet connection

### Google Sign-In Not Working

**Issue:** Google Sign-In fails  
**Solution:**
- Ensure Google account is active
- Grant all permissions
- Try email/password registration instead
- Check internet connection
